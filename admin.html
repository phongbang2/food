Quản trị dữ liệu

Đây là cổng mở bảng duyệt quán. Mỗi thay đổi phải được kiểm tra rồi mới ghi vào tab HCM.

Thiết lập một lần

Trong Google Apps Script, triển khai file quản trị trong thư mục admin/ thành Web app. Sau khi cấp quyền Google, dán URL Web app vào ô dưới đây.

Web app nên đặt “Execute as: Me” và giới hạn người có quyền truy cập. Không chia sẻ URL quản trị công khai.
Mở file gốc

Cài bảng quản trị

  1. Mở script.google.com bằng đúng tài khoản có quyền sửa file.
  2. Tạo project mới, dán admin/Code.gs và tạo file HTML tên Admin, sau đó dán admin/Admin.html.
  3. Deploy → New deployment → Web app → chạy bằng tài khoản của bạn → cấp quyền khi Google yêu cầu.
  4. Dán URL Web app ở trên. Lần sau chỉ cần bấm “Quản trị dữ liệu”.

Không cần Google Maps API. Việc thêm “quán hot” vẫn cần nguồn review/link để tránh tự bịa địa chỉ hoặc giờ mở cửa.
